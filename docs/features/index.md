# docuMINE Feature Roadmap

> Expanding docuMINE from document comparison into the AI-powered command center for independent insurance agents.

**IMPORTANT:** These are all **features within the docuMINE application** - not separate products. All features share the same codebase, authentication, database, and deployment.

"docuMINE is the AI-powered command center for independent insurance agents."

## Feature Details

### AI Buddy (Phase 1)
The engagement hook and foundation for docuMINE's expanded capabilities.

- ChatGPT-style UI with projects/workspaces
- Personalized learning (preferences, behavior, deep context)
- Agency admin panel with guardrails
- Per-seat pricing model

**Docs:** [ai-buddy/](./ai-buddy/)

### Custom Reporting (Phase 2)
The data moat that creates stickiness.

- Upload Excel/CSV commission data
- AI normalizes columns automatically
- Natural language queries about your business
- Chart and export generation

**Docs:** [reporting/](./reporting/)

### Quoting (Phase 3 & 4)
The holy grail for insurance agents.

- Phase 3: Clipboard helper (no extension required)
- Phase 4: AI-powered background quoting (any carrier)

**Docs:** [quoting/](./quoting/)

### Research
Technical research supporting new feature development.

**Docs:** [research/](./research/)
