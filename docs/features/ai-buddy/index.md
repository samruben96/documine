# AI Buddy (Phase 1)

> Personalized AI assistant that learns about each user over time, with agency-level guardrails for compliance and control.

## Status: Planning

## Overview

AI Buddy is the engagement hook and foundation for docuMINE's expanded capabilities. It provides:

- **ChatGPT-style interface** with sidebar for conversation history
- **Projects/Workspaces** organized by client accounts
- **Personalized learning** across three tiers (explicit, behavioral, deep)
- **Agency admin panel** with guardrails, permissions, and audit trails

## Key Features

### For Agents (Daily Users)
- Conversational AI with memory
- Project-based organization (by client account)
- Document attachment and context
- Quick actions based on usage patterns

### For Agency Admins
- Guardrails configuration (topics, carriers, compliance)
- Per-producer permissions
- Audit trail of all AI conversations
- Usage analytics
